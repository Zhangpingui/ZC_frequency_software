# Mock frequency-assignment datasets

本目录保存频率指配软件使用的人工模拟测试数据。

这些数据仅用于软件功能测试、性能测试和界面演示，
不包含真实设备、真实任务或其他敏感信息。

## links_100.xlsx

### 数据规模

- 通信链路数量：100
- 用频设备数量：200
- 链路编号：L001～L100
- 设备编号：D001～D200
- 坐标区域：0～100 km
- 中心频率范围：2.40～2.49 GHz
- 带宽：20 MHz
- 发射功率：30 dBm

### 数据字段

- `link_id`：链路编号
- `tx_id`：发射设备编号
- `tx_x_km`：发射设备 X 坐标
- `tx_y_km`：发射设备 Y 坐标
- `rx_id`：接收设备编号
- `rx_x_km`：接收设备 X 坐标
- `rx_y_km`：接收设备 Y 坐标
- `frequency_ghz`：中心频率
- `bandwidth_mhz`：信号带宽
- `tx_power_dbm`：发射功率

### 测试参考

按照当前冲突判定逻辑：

- 空间干扰阈值：10 km
- 保护频差：20 MHz

预计检测约 195 对冲突。

其中：

- 频谱重叠或边界接触：141 对
- 邻频保护间隔不足：54 对

如果后续修改冲突计算规则，应重新核验预期结果。

### 使用方法

1. 启动 Streamlit 应用；
2. 进入“数据建模”页面；
3. 上传 `links_100.xlsx`；
4. 点击“解析并加载文件”；
5. 进入“冲突计算”页面；
6. 设置空间阈值和保护频差；
7. 执行冲突检测和算法测试。
